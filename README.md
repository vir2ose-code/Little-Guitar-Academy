# 🎸 Little Guitar Academy

Eine interaktive, mehrsprachige Smartphone-Anwendung für kleine Gitarrenhelden (6-8 Jahre).

## ✨ Features
- **Interaktiver Saiten-Stimmer**: Präzise Audio-Unterstützung für die perfekte Stimmung.
- **Akkord-Lektionen**: Magisches Vollbild-Carousel mit authentischen Audio-Aufnahmen (C-DUR, D-DUR, etc.).
- **4 Sprachen**: Vollständige Lokalisierung in Deutsch, Englisch, Polnisch und Spanisch.
- **Premium Design**: Edle Holz-Optik, goldene Branding-Elemente und flüssige Animationen.
- **Privacy First**: Inklusive interaktivem Datenschutz-Modal und kindersicherem Design.

## 🛠 Tech-Stack
- **HTML5 & Vanilla CSS3**: Hochperformantes, schlankes UI ohne Overkill.
- **Modern JavaScript (ES6+)**: Dynamisches Carousel-System und Audio-Engine.
- **Vercel/GitHub Ready**: Optimiert für schnelles Deployment.

## 🚀 Installation & Start
Einfach die `index.html` in einem modernen Browser öffnen oder direkt via Vercel/GitHub Pages hosten.
